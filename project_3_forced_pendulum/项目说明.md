# 项目3：受驱单摆的非线性动力学

## 项目概述

本项目研究受外力驱动的非线性单摆系统，探索其丰富的动力学行为，包括周期运动、准周期运动、混沌运动和共振现象。通过数值求解和可视化分析，深入理解非线性振动系统的复杂性。

## 数学模型

### 受驱阻尼单摆方程

考虑受到周期外力驱动和阻尼作用的单摆，其运动方程为：

$$d^2\theta/dt² + \gamma(d\theta/dt) + \omega_0^2\sin(\theta) = f_0\cos(\omega t)$$


其中：
-  $\theta$：摆角（弧度）
-  $\gamma$：阻尼系数
-  $\omega_0$：固有频率 ( $\omega_0 =\sqrt{g/L}$)
-  $f_0$：驱动力幅度
-  $\omega$：驱动频率
-  $t$：时间

### 状态空间表示

将二阶微分方程转化为一阶方程组：

$$\begin{align*}
d\theta/dt=\omega_θ\\
dω_θ/dt=-\omega_0^2\sin(\theta) + f_0\cos(\omega t)
\end{align*}$$


状态向量：**y** = [θ, ω_θ]ᵀ

### 无量纲化

引入无量纲时间 τ = ω₀t，方程变为：

```
d²θ/dτ² + (γ/ω₀)(dθ/dτ) + sin(θ) = (f₀/ω₀²)cos((ω/ω₀)τ)
```

定义无量纲参数：
- β = γ/ω₀：无量纲阻尼系数
- F = f₀/ω₀²：无量纲驱动力
- Ω = ω/ω₀：频率比
